# NSTA V-1 Feature List

## Student Features

1.  **Smart Video Lectures** - High-quality educational content.
    *   *Standard Videos*: Available for Free/Basic users.
    *   *4K / Long Duration / Animated Videos*: **UPGRADABLE BY PREMIUM SUBSCRIPTION**
2.  **PDF Notes Library** - Comprehensive study material.
    *   *Basic Notes*: Available for Free users.
    *   *Detailed / Ultra Notes*: **UPGRADABLE BY PREMIUM SUBSCRIPTION**
3.  **MCQ Practice Zone** - Chapter-wise quizzes.
    *   *Free Limit*: 50 MCQs per session.
    *   *Unlimited Practice*: **UPGRADABLE BY PREMIUM SUBSCRIPTION** (100+ questions)
4.  **Weekly Tests** - Scheduled exams for self-assessment.
5.  **Daily Challenge** - 15-minute speed test (100 Qs). Score 90%+ to win 1 Month Free Subscription.
6.  **Live Leaderboard** - Compete globally. Top 3 ranks get Gold Theme.
7.  **Engagement Rewards** - Earn coins or subscriptions by studying (Time-based unlocking).
8.  **Universal Chat** - Public discussion room.
9.  **Private Admin Support** - Direct line to teachers/admin.
    *   *Priority Support*: **UPGRADABLE BY PREMIUM SUBSCRIPTION**
10. **Spin Wheel Game** - Daily luck-based rewards (Coins).
    *   *Daily Limit*: 2 Spins (Free).
    *   *Unlimited/Higher Limit*: **UPGRADABLE BY PREMIUM SUBSCRIPTION**
11. **Credit System** - Earn and spend virtual currency for content.
12. **Subscription Plans** - Weekly, Monthly, Yearly, Lifetime tiers.
13. **Profile Customization** - Edit details and avatars.
14. **Study Timer** - Tracks daily activity.
15. **Daily Goal Tracker** - Pop-up tracker for 3-hour study target.
16. **User Inbox** - Receive messages and gifts from Admin.
17. **Performance History** - Track test scores and progress.
18. **Dark/Light Mode** - Visual comfort settings.
19. **Responsive Design** - Works on Mobile and Tablet.
20. **Offline Capabilities** - **UPGRADABLE BY PREMIUM SUBSCRIPTION** (Video Downloads).
21. **Passwordless Login** - Secure, hassle-free access.
22. **Custom Subjects** - Dynamic subject support beyond core curriculum.
23. **Featured Shortcuts** - Quick access to key chapters on Home Screen.
24. **AI Question Generator** - Auto-generate MCQs from topic.
25. **Daily Login Bonus** - Automatic credit rewards.
26. **Theme Customization** - Dynamic App Colors.
    *   *Basic Blue*: Default.
    *   *Premium Purple / Gold*: **UPGRADABLE BY PREMIUM SUBSCRIPTION** or Leaderboard Rank.
27. **Floating Dock** - Quick navigation menu.
28. **Marquee Announcements** - Real-time scrolling updates.
29. **Detailed Marksheet** - Post-test analysis with time per question.
30. **Question Analysis** - Correct vs Wrong breakdown.

## Admin Features

31. **Admin Dashboard** - Central control panel.
32. **User Management** - View, Edit, Delete, Impersonate users.
33. **Subscription Management** - Grant Free/Paid subscriptions with auto-duration.
34. **Content Manager** - Upload PDF links, Video links, create MCQs.
35. **Bulk Upload** - Fast content entry via copy-paste.
36. **Syllabus Management** - Add/Edit Chapters and Subjects.
37. **Settings Configuration** - Toggle features, ads, security, payments.
38. **Gift Code Generator** - Create redeemable codes for credits.
39. **Sub-Admin Management** - Create staff accounts with granular permissions (30+ controls).
40. **Activity Log** - Track system usage and admin actions.
41. **Database Viewer** - Direct access to raw data.
42. **Recycle Bin** - Restore deleted users or content (90 days retention).
43. **Deployment Tools** - Download source code for updates.
44. **Universal Video Playlist** - Manage featured videos for all users.
45. **Role Management** - Assign Admin/Sub-Admin/Student roles.
46. **Ban System** - Restrict user access.
47. **Impersonation Mode** - View app as a specific student.
48. **Revenue Tracking** - View subscription sales reports.
49. **Watermark Configuration** - Customize video player overlay.
50. **Feature Popup Config** - Edit Free vs Premium feature lists.

## Content & Academic Features

51. **Board Support** - CBSE, BSEB, etc.
52. **Class Management** - Class 6 to 12 + Competition.
53. **Stream Support** - Science, Commerce, Arts.
54. **Multi-Language Support** - English & Hindi interface.
55. **HTML Rich Text Notes** - Interactive reading material.
56. **AI Visual Notes** - Image-based study aids.
57. **Sequential Unlocking** - Require MCQ completion to unlock next chapter (Optional).
58. **Video Quality Control** - Player settings.
59. **Playback Speed Control** - 0.5x to 2x speed.
60. **Picture-in-Picture Mode** - Watch while multitasking.

## Monetization & Security

61. **Payment Gateway Integration** - UPI / Manual payment support.
62. **Store** - Buy credits or packages.
63. **Startup Ad** - Promotional popup on launch.
64. **Security System** - Prevent multiple logins (Session management).
65. **PDF Watermarking** - Protect content privacy.
